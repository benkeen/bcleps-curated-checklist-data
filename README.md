## BC Leps Curated Checklist data

This repo houses the raw data used in the [Curated Species](https://bcleps.weebly.com/curated-species.html) page. It's generated by the https://github.com/benkeen/inat-curated-species-list application, allowing specific curator to generate a tailored list of the species found in the project. See that repo for more information.
